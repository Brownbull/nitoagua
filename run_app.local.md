# nitoagua - Local Development

**Port:** 3005

## Start App

```bash
npm run dev
```

Open http://localhost:3005

## Setup (First Time)

```bash
npm install
cp .env.local.example .env.local
# Edit .env.local with your Supabase credentials
```

## Build

```bash
npm run build
npm run start
```

## Lint

```bash
npm run lint
```
