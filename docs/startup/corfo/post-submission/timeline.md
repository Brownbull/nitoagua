## Qué Esperar Durante la Evaluación

### Proceso de Evaluación (19 dic 2025 - 5 mar 2026)

El proceso de evaluación de CORFO Semilla Inicia incluye:

1. **Admisibilidad** - Verificación de requisitos formales
   - ¿Postulante cumple requisitos de edad, residencia?
   - ¿Proyecto está dentro de los focos permitidos?
   - ¿Documentación completa?

2. **Evaluación Técnica** - Panel de evaluadores
   - **Innovación (40%)** - ¿Es realmente innovador? ¿Diferenciación clara?
   - **Escalabilidad (30%)** - ¿Tiene potencial de crecimiento nacional/internacional?
   - **Equipo (30%)** - ¿Capacidad de ejecutar el proyecto?

3. **Video Review** - Evaluación del pitch de 40 segundos
   - Claridad del mensaje
   - Presentación del producto/servicio
   - Comunicación de la innovación

### Posible Entrevista

Si el proyecto pasa la evaluación inicial, es posible que se solicite una entrevista. Ver documento `interview-prep.md` para preparación.

## Si Adjudicamos: Próximos Pasos

### Firma de Contrato (Marzo - Abril 2026)

1. Recibir notificación oficial de CORFO
2. Revisar términos del contrato
3. Preparar documentación requerida:
   - Constitución de empresa (si aún no existe)
   - Cuenta bancaria empresarial
   - Documentos legales adicionales
4. Firmar contrato con CORFO

### Entrega del Beneficio (Mayo 2026)

2. Iniciar ejecución formal del proyecto
